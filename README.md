# github-explorer

Application made for learning purposes.
It can search for github repositories and list the open issues.

### Techs
- ReactJS
- TypeScript
- Axios
- Styled-Components
- React-Router-Dom
- React-Icons
- Polished
- Eslint
- Prettier
